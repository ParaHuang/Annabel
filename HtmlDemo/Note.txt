2.style of background
background-color	->	set up the background color,, the value can be an English word repsent color, or a hexadecimal number(#ff00ff)
background-image	->	set up the background image
background			->	color + image
background-size		->	adjust the size of image to the size of lable
						value:
							1)contain:equal scaling the picture, till the picture is fully display in label, if there are extra space, repeat the picture
							2)cover:equal scaling the picture, till the picture is fully covered the label, the picture might be cut off fromt he edge
							3)100% 100%： make the picture fully fit in lable

background-repeat	->	value:
							1)no-repeat
							2)repeat, the default way
							3)repeat-x, repeat-y
